About StreamWhere StreamWhere is nothing more than a demonstration of Ionic framework for my Universty dissertation. More spesificly the usege/quality/functionalty of the applications generated by the framework.

The critical question that StreamWhere aimed to answer is ' Can a novice (Noob if you will) programmer produce a best-of bread mobile application using Ionic 4 framework?' Two different API's are used to acquire the content.

OMDB:

Open Movie Database API is used to produce media content and the media descriptions. You can find more information about OMDB here:
http://www.omdbapi.com/

Utelly:

Provides the avilibilly data that is shown to the user in the final page of StreamWhere. You can find more information about Utelly API here:
https://rapidapi.com/utelly/api/utelly
